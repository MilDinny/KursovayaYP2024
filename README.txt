Приложение для хранения информации о проектах
Для начала использования приложения необходимо лишь 1 простое действие - запуск установщика (файл main.exe)
После запуска вы увидите окно авторизации. Если еще нет аккаунта, требуется зарегестрироваться.
После успешного входа, вы можете ввести информацию о проекте и внести его в свой список.
Все данные сохраняются в базе данных, поэтому, когда вы в следующий раз захотите воспользоваться приложением,
все данные будут сохранены. Также в приложении присутствуют напоминания о сроках сдачи, что поможет вам ничего не забыть
