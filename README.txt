TaskBoard Application


Описание:
TaskBoard - это консольное приложение для управления проектами, реализованное на языке Python с использованием библиотеки Tkinter для графического интерфейса. Приложение позволяет пользователям регистрироваться, входить в систему, создавать и управлять проектами, а также экспортировать проекты в CSV файл. Администраторы могут управлять пользователями.


Возможности:
- Регистрация пользователей
- Вход в систему
- Просмотр списка пользователей (для администратора)
- Удаление аккаунта пользователя
- Создание, просмотр и удаление проектов
- Экспорт проектов в CSV файл
- Управление пользователями (для администратора)


Зависимости:
- Для работы приложения необходимо установить следующие зависимости:
	-- Python 3.x
	-- Tkinter (обычно входит в стандартную библиотеку Python)


Установка:
- Клонируйте репозиторий или загрузите архив с проектом.
- Установите необходимые зависимости:
	bash
	-- pip install tk


Использование:
- Запустите приложение
	bash
	-- python main.py
- Следуйте инструкциям на экране для регистрации нового пользователя или входа в систему.
- Используйте личный кабинет для создания и управления проектами.
